# Context

<CodePreview data={contextTypePreview} /> is a context object created via <CodePreview data={createContextPreview} />.
The value returned by `createContext` is itself a **Context component**. You use it as a JSX component with a `value` prop:

```tsx
<ThemeContext value={themeState}>
  <App />
</ThemeContext>
```

Descendant components call `useContext(ThemeContext)` to read the nearest matching context's value. 
If no matching context is found, `useContext` returns the default value passed to `createContext`.

### Example

<div>
  <div className="not-prose">
    <TabGroup
      items={["ThemeContext", "App", "Button"]}
      onSelect={(value) => (selectedTab.value = value)}
      value={selectedTab.value}
      itemSuffix={(item) => `.${siteCodeLang.value}${item === "ThemeContext" ? "" : "x"}`}
    />
    <CopyInnerText>
      <Derive from={{siteCodeLang, selectedTab}}>
        {({siteCodeLang, selectedTab}) => {
          return match(siteCodeLang)
            .with("js", () => { 
              return match(selectedTab)
                .with("App", () => <ExampleApp />)
                .with("ThemeContext", () => <ExampleThemeContext />)
                .else(() => <ExampleButton />)
            })
            .else(() => {
              return match(selectedTab)
                .with("App", () => <ExampleApp />)
                .with("ThemeContext", () => <ExampleThemeContextTS />)
                .else(() => <ExampleButton />)
            })
        }}
      </Derive>
    </CopyInnerText>
  </div>
</div>
